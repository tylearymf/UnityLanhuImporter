# LanhuImporter - 蓝湖 UI 转换器

将[蓝湖](https://lanhuapp.com)设计稿一键转换为 Unity UGUI 层级结构的 Editor 工具。

## 前置依赖

| 依赖 | 版本 | 说明 |
|------|------|------|
| Unity | 2020.3+ | 需要支持 C# 9 语法 |
| **Odin Inspector** | 3.x+ | **付费插件**，用于编辑器窗口美化（搜索栏、Box 容器等）。需自行购买：[https://odininspector.com](https://odininspector.com) |

> **注意**：本仓库仅包含 `LanhuImporter.cs` 单文件，不包含 Odin Inspector。请确保项目中已安装 Odin Inspector 后再导入本工具。

## 安装

将 `LanhuImporter.cs` 放入 Unity 项目的 `Assets/Editor/` 目录下（或其任意子目录）即可。

```
YourProject/
└── Assets/
    └── Editor/
        └── Tools/
            └── LanhuImporter.cs
```

## 使用方法

### 1. 打开工具

菜单栏 → `Tools` → `蓝湖UI转换器`

### 2. Tab 页说明

工具窗口分为两个 Tab：

#### Tab 1 - 转换设置

| 区域 | 说明 |
|------|------|
| **数据源设置** | 选择导入模式（URL / 本地文件），配置蓝湖 URL 和 Cookie |
| **比例设置** | 设置缩放倍数或按指定宽度自动计算缩放 |
| **模块设置** | 设置当前界面模块路径和精灵导入目录后缀 |

#### Tab 2 - 路径配置

| 区域 | 说明 |
|------|------|
| **公共模块目录** | 添加多个公共精灵资源搜索目录 |
| **字体目录** | 添加多个字体资源搜索目录 |
| **其他设置** | 配置外部 UI 目录和图片缓存目录 |

### 3. 操作流程

#### 方式一：通过蓝湖项目 URL 导入

1. 在「导入模式」中选择 `JsonUrl`
2. 粘贴蓝湖项目页面 URL（包含 `project_id` 参数的链接）
3. 填写蓝湖 Cookie（从浏览器开发者工具中获取，点击「保存」持久化）
4. 点击「获取项目界面列表并选中目标界面」
5. 在列表中点击选中目标界面（支持搜索、按名称/创建时间/更新时间排序）
6. 在 Tab 2 中配置好路径（首次使用时需配置）
7. 回到 Tab 1，点击「开始转换并生成界面」

#### 方式二：通过本地 JSON 文件导入

1. 在「导入模式」中选择 `JsonFile`
2. 点击「浏览」选择导出的 JSON 文件
3. 配置好路径和比例设置
4. 点击「开始转换并生成界面」

#### 方式三：通过直接 JSON URL 导入

1. 在「导入模式」中选择 `JsonUrl`
2. 粘贴包含 `SketchJSONURL` 的直接 JSON 链接
3. 点击「开始转换并生成界面」

### 4. Cookie 获取方法

1. 在浏览器中登录 [蓝湖](https://lanhuapp.com)
2. 打开开发者工具（F12）→ Application → Cookies
3. 复制 `lanhuapp.com` 下的完整 Cookie 字符串

## 功能特性

- **双版本支持**：自动识别蓝湖 V1（PSD）和 V2（Sketch）数据格式
- **界面列表**：三列显示（名称、创建时间、更新时间），支持搜索和排序
- **预览图**：自动下载蓝湖界面预览图并作为首个子节点挂载到生成的预制体中
- **精灵匹配**：按名称自动搜索当前模块、公共模块和外部 UI 目录中的精灵资源
- **外部资源拷贝**：从项目外目录自动拷贝匹配的精灵到指定输出目录
- **字体匹配**：支持精确匹配和模糊匹配字体资源
- **富文本**：自动处理多色、多样式的富文本内容
- **九宫格**：自动识别已设置 Border 的精灵并启用 Sliced 模式
- **坐标转换**：自动将蓝湖坐标系转换为 Unity 锚点坐标系
- **Transform 优化**：自动将位置、尺寸、旋转取整

## 配置持久化

以下配置通过 `EditorPrefs` 自动持久化，重启 Unity 后保留：

- 公共模块目录列表
- 字体目录列表
- 蓝湖 Cookie
- 外部 UI 目录
- 精灵导入目录后缀
- 图片缓存目录

## 注意事项

- 场景中需要存在 Canvas，若不存在会自动创建
- 精灵名称需为纯英文/数字/下划线格式（`^[a-zA-Z0-9_]+$`）才会触发自动匹配
- 蓝湖 Cookie 有效期有限，过期后需重新获取
- 预览图下载使用同步请求，大图可能导致编辑器短暂卡顿
- 时间显示为北京时间（UTC+8）
